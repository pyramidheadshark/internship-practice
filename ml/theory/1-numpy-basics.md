**NumPy** (Numerical Python) - —ç—Ç–æ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –¥–ª—è —á–∏—Å–ª–µ–Ω–Ω—ã—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏–π –Ω–∞ Python. –û–Ω–∞ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∫—É –±–æ–ª—å—à–∏—Ö, –º–Ω–æ–≥–æ–º–µ—Ä–Ω—ã—Ö –º–∞—Å—Å–∏–≤–æ–≤ –∏ –º–∞—Ç—Ä–∏—Ü, –∞ —Ç–∞–∫–∂–µ –∫–æ–ª–ª–µ–∫—Ü–∏—é –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π –¥–ª—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–π —Ä–∞–±–æ—Ç—ã —Å —ç—Ç–∏–º–∏ –º–∞—Å—Å–∏–≤–∞–º–∏. –í –º–∞—à–∏–Ω–Ω–æ–º –æ–±—É—á–µ–Ω–∏–∏ NumPy –Ω–µ–æ–±—Ö–æ–¥–∏–º, –ø–æ—Å–∫–æ–ª—å–∫—É –æ–Ω —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç –æ—Å–Ω–æ–≤—É –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∏ –º–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞–Ω–∏—è –¥–∞–Ω–Ω—ã–º–∏, –æ—Å–æ–±–µ–Ω–Ω–æ –≤–µ–∫—Ç–æ—Ä–∞–º–∏ –∏ –º–∞—Ç—Ä–∏—Ü–∞–º–∏, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –æ—Å–Ω–æ–≤–Ω—ã–º–∏ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–º–∏ —Å—É—â–Ω–æ—Å—Ç—è–º–∏ –≤ –∞–ª–≥–æ—Ä–∏—Ç–º–∞—Ö ML.

## –ß—Ç–æ —Ç–∞–∫–æ–µ –º–∞—Å—Å–∏–≤—ã NumPy? ü§î

–í –æ—Å–Ω–æ–≤–µ NumPy –ª–µ–∂–∏—Ç **ndarray**, –∏–ª–∏ N-–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤ (**N-dimensional array**). –ü—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç—Å—è –∫–∞–∫ —Å–µ—Ç–∫–∞ –∑–Ω–∞—á–µ–Ω–∏–π –æ–¥–Ω–æ–≥–æ –∏ —Ç–æ–≥–æ –∂–µ —Ç–∏–ø–∞, –∏–Ω–¥–µ–∫—Å–∏—Ä—É–µ–º–∞—è –∫–æ—Ä—Ç–µ–∂–µ–º –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã—Ö —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª.

**–ö–ª—é—á–µ–≤—ã–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ –º–∞—Å—Å–∏–≤–æ–≤ NumPy:**

*   **–û–¥–Ω–æ—Ä–æ–¥–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö (Homogeneous Data Type):** –í—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –º–∞—Å—Å–∏–≤–µ NumPy –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –æ–¥–Ω–æ–≥–æ –∏ —Ç–æ–≥–æ –∂–µ —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ, —á–∏—Å–ª–æ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π). –≠—Ç–∞ –æ–¥–Ω–æ—Ä–æ–¥–Ω–æ—Å—Ç—å –¥–µ–ª–∞–µ—Ç –º–∞—Å—Å–∏–≤—ã NumPy —Ç–∞–∫–∏–º–∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–º–∏ –¥–ª—è —á–∏—Å–ª–µ–Ω–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π.
*   **–ú–Ω–æ–≥–æ–º–µ—Ä–Ω–æ—Å—Ç—å (Multidimensional):** –ú–∞—Å—Å–∏–≤—ã NumPy –º–æ–≥—É—Ç –±—ã—Ç—å 1-–º–µ—Ä–Ω—ã–º–∏ (–≤–µ–∫—Ç–æ—Ä—ã), 2-–º–µ—Ä–Ω—ã–º–∏ (–º–∞—Ç—Ä–∏—Ü—ã) –∏–ª–∏ –∏–º–µ—Ç—å –µ—â–µ –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–µ –∏–∑–º–µ—Ä–µ–Ω–∏—è (—Ç–µ–Ω–∑–æ—Ä—ã).
*   **–§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Ä–∞–∑–º–µ—Ä (Fixed Size):** –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç —Å–ø–∏—Å–∫–æ–≤ Python, –º–∞—Å—Å–∏–≤—ã NumPy –∏–º–µ—é—Ç —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Ä–∞–∑–º–µ—Ä –ø—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏. –ò–∑–º–µ–Ω–µ–Ω–∏–µ —Ä–∞–∑–º–µ—Ä–∞ –æ–±—ã—á–Ω–æ –≤–∫–ª—é—á–∞–µ—Ç —Å–æ–∑–¥–∞–Ω–∏–µ –Ω–æ–≤–æ–≥–æ –º–∞—Å—Å–∏–≤–∞.
*   **–≠—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ (Efficient Operations):** NumPy —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω –Ω–∞ C –∏ Fortran, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–∞—Å—Å–∏–≤–∞–º–∏ –Ω–∞–º–Ω–æ–≥–æ –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º –∏—Ç–µ—Ä–∞—Ü–∏–∏ –ø–æ —Å–ø–∏—Å–∫–∞–º Python, –æ—Å–æ–±–µ–Ω–Ω–æ –¥–ª—è –±–æ–ª—å—à–∏—Ö –Ω–∞–±–æ—Ä–æ–≤ –¥–∞–Ω–Ω—ã—Ö.

## –ë–∞–∑–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–∞—Å—Å–∏–≤–∞–º–∏ NumPy ‚ûï‚ûñ‚úñÔ∏è‚ûó

### –°–æ–∑–¥–∞–Ω–∏–µ –º–∞—Å—Å–∏–≤–æ–≤ NumPy (Creating NumPy Arrays)

–ú–∞—Å—Å–∏–≤—ã NumPy –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞–≤–∞—Ç—å –∏–∑ —Å–ø–∏—Å–∫–æ–≤ –∏–ª–∏ –∫–æ—Ä—Ç–µ–∂–µ–π Python:

### –ë–∞–∑–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–∞—Å—Å–∏–≤–∞–º–∏ (Basic Array Operations)

NumPy –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –≤—ã–ø–æ–ª–Ω—è—Ç—å –ø–æ—ç–ª–µ–º–µ–Ω—Ç–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏, –∞ —Ç–∞–∫–∂–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –º–µ–∂–¥—É –º–∞—Å—Å–∏–≤–∞–º–∏ –∏ —Å–∫–∞–ª—è—Ä–∞–º–∏.

```python

# –°–∫–∞–ª—è—Ä–Ω–æ–µ —Å–ª–æ–∂–µ–Ω–∏–µ (Scalar addition)

```
