## –ß—Ç–æ —Ç–∞–∫–æ–µ –º–∞—Å—Å–∏–≤—ã NumPy? ü§î

–í –æ—Å–Ω–æ–≤–µ NumPy –ª–µ–∂–∏—Ç –æ–±—ä–µ–∫—Ç **ndarray** ‚Äî N-–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤. –≠—Ç–æ —Å–µ—Ç–∫–∞ –∑–Ω–∞—á–µ–Ω–∏–π –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞, –∫ –∫–æ—Ç–æ—Ä—ã–º –º–æ–∂–Ω–æ –æ–±—Ä–∞—â–∞—Ç—å—Å—è –ø–æ –∏–Ω–¥–µ–∫—Å–∞–º.

**–ö–ª—é—á–µ–≤—ã–µ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:**
- **–û–¥–Ω–æ—Ä–æ–¥–Ω–æ—Å—Ç—å:** –í—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `float` –∏–ª–∏ `int`), —á—Ç–æ —É—Å–∫–æ—Ä—è–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏—è.
- **–ú–Ω–æ–≥–æ–º–µ—Ä–Ω–æ—Å—Ç—å:** –ú–∞—Å—Å–∏–≤—ã –º–æ–≥—É—Ç –±—ã—Ç—å 1D (–≤–µ–∫—Ç–æ—Ä—ã), 2D (–º–∞—Ç—Ä–∏—Ü—ã) –∏–ª–∏ N-–º–µ—Ä–Ω—ã–º–∏ (—Ç–µ–Ω–∑–æ—Ä—ã).
- **–§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Ä–∞–∑–º–µ—Ä:** –†–∞–∑–º–µ—Ä –∑–∞–¥–∞–µ—Ç—Å—è –ø—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏, —á—Ç–æ –æ—Ç–ª–∏—á–∞–µ—Ç –∏—Ö –æ—Ç —Å–ø–∏—Å–∫–æ–≤ Python.
- **–≠—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å:** –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –Ω–∞ C –¥–µ–ª–∞–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º —Å –æ–±—ã—á–Ω—ã–º–∏ —Å–ø–∏—Å–∫–∞–º–∏.

**–ü—Ä–∏–º–µ—Ä —Å—Ç—Ä—É–∫—Ç—É—Ä—ã (–∞–Ω–∞–ª–æ–≥–∏—è):**
- 1D –º–∞—Å—Å–∏–≤ `[1, 2, 3]` ‚Äî –∫–∞–∫ —Å–ø–∏—Å–æ–∫ –ø–æ–∫—É–ø–æ–∫.
- 2D –º–∞—Å—Å–∏–≤ `[[1, 2], [3, 4]]` ‚Äî –∫–∞–∫ —Ç–∞–±–ª–∏—Ü–∞ (—Å—Ç—Ä–æ–∫–∏ ‚Äî –æ–±—ä–µ–∫—Ç—ã, —Å—Ç–æ–ª–±—Ü—ã ‚Äî –∏—Ö —Å–≤–æ–π—Å—Ç–≤–∞).

## –ë–∞–∑–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–∞—Å—Å–∏–≤–∞–º–∏ NumPy ‚ûï‚ûñ‚úñÔ∏è‚ûó

–£—Å—Ç–∞–Ω–æ–≤–∏–º NumPy (–µ—Å–ª–∏ –µ—â–µ –Ω–µ —Å–¥–µ–ª–∞–Ω–æ):

### –°–æ–∑–¥–∞–Ω–∏–µ –º–∞—Å—Å–∏–≤–æ–≤

# 1D –º–∞—Å—Å–∏–≤ (–≤–µ–∫—Ç–æ—Ä) ‚Äî —Å—Ç—Ä–æ–∫–∞ —á–∏—Å–µ–ª

# 2D –º–∞—Å—Å–∏–≤ (–º–∞—Ç—Ä–∏—Ü–∞) ‚Äî —Ç–∞–±–ª–∏—Ü–∞ —á–∏—Å–µ–ª

### –û–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–∞—Å—Å–∏–≤–∞–º–∏

# –ü–æ—ç–ª–µ–º–µ–Ω—Ç–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ (–∫–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç—Å—è –æ—Ç–¥–µ–ª—å–Ω–æ)

## –í–≤–µ–¥–µ–Ω–∏–µ –≤ –≤–µ–∫—Ç–æ—Ä—ã –∏ –º–∞—Ç—Ä–∏—Ü—ã üìê

–í ML:
- **–í–µ–∫—Ç–æ—Ä—ã (1D –º–∞—Å—Å–∏–≤—ã):** –≠—Ç–æ —Ç–æ—á–∫–∏ –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ. –ù–∞–ø—Ä–∏–º–µ—Ä, `[—Ä–æ—Å—Ç, –≤–µ—Å]` –æ–ø–∏—Å—ã–≤–∞–µ—Ç —á–µ–ª–æ–≤–µ–∫–∞.
- **–ú–∞—Ç—Ä–∏—Ü—ã (2D –º–∞—Å—Å–∏–≤—ã):** –≠—Ç–æ –Ω–∞–±–æ—Ä—ã —Ç–æ—á–µ–∫. –ù–∞–ø—Ä–∏–º–µ—Ä, —Ç–∞–±–ª–∏—Ü–∞, –≥–¥–µ —Å—Ç—Ä–æ–∫–∏ ‚Äî –æ–±—Ä–∞–∑—Ü—ã, –∞ —Å—Ç–æ–ª–±—Ü—ã ‚Äî –ø—Ä–∏–∑–Ω–∞–∫–∏.

**–ü—Ä–∏–º–µ—Ä –∏–∑ ML:** –î–∞—Ç–∞—Å–µ—Ç —Å —Ç—Ä–µ–º—è –ª—é–¥—å–º–∏:
```

 [165, 60],  # –≤—Ç–æ—Ä–æ–≥–æ

NumPy –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å –æ–ø–µ—Ä–∞—Ü–∏–∏ –ª–∏–Ω–µ–π–Ω–æ–π –∞–ª–≥–µ–±—Ä—ã:  
```

print("–ú–∞—Ç—Ä–∏—á–Ω–æ–µ —É–º–Ω–æ–∂–µ–Ω–∏–µ:\n", np.dot(A, B))

# –¢—Ä–∞–Ω—Å–ø–æ–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ (–º–µ–Ω—è–µ–º —Å—Ç—Ä–æ–∫–∏ –∏ —Å—Ç–æ–ª–±—Ü—ã)

```

–≠—Ç–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ –Ω—É–∂–Ω—ã, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏, –≥–¥–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ = –ø—Ä–∏–∑–Ω–∞–∫–∏ √ó –≤–µ—Å–∞.

## –ü—Ä–∏–º–µ—Ä –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –≤ ML

–ü—Ä–µ–¥—Å—Ç–∞–≤–∏–º –¥–∞–Ω–Ω—ã–µ –∏ —Å–¥–µ–ª–∞–µ–º –ø—Ä–æ—Å—Ç–æ–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ:  
```
