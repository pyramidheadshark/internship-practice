## 3. üìà Matplotlib - –û—Å–Ω–æ–≤–∞ –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏–∏ –≤ Python

**Matplotlib** ‚Äî —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –¥–ª—è –≥—Ä–∞—Ñ–∏–∫–æ–≤ –≤ Python. –û–Ω–∞ –¥–∞–µ—Ç **–ø–æ–ª–Ω—ã–π –∫–æ–Ω—Ç—Ä–æ–ª—å** –Ω–∞–¥ –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏–µ–π.

* **–ú–æ–¥—É–ª—å:** `import matplotlib.pyplot as plt`.
* **–û—Å–Ω–æ–≤–Ω—ã–µ —Ç–∏–ø—ã –≥—Ä–∞—Ñ–∏–∫–æ–≤:**
  - **–õ–∏–Ω–µ–π–Ω—ã–µ (`plt.plot`):** –¢—Ä–µ–Ω–¥—ã, –≤—Ä–µ–º–µ–Ω–Ω—ã–µ —Ä—è–¥—ã.
  - **–¢–æ—á–µ—á–Ω—ã–µ (`plt.scatter`):** –ó–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏, –∫–ª–∞—Å—Ç–µ—Ä—ã, –≤—ã–±—Ä–æ—Å—ã.
  - **–ì–∏—Å—Ç–æ–≥—Ä–∞–º–º—ã (`plt.hist`):** –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è (–∞–Ω–∞–ª–æ–≥–∏—è: –¥–µ–ª–∏–º –¥–∞–Ω–Ω—ã–µ –Ω–∞ "–∫–æ—Ä–∑–∏–Ω—ã", –∫–∞–∫ –ª—é–¥–µ–π –ø–æ —Ä–æ—Å—Ç—É).
  - **–ö–æ—Ä–æ–±—á–∞—Ç—ã–µ (`plt.boxplot`):** –ö–≤–∞—Ä—Ç–∏–ª–∏, –≤—ã–±—Ä–æ—Å—ã.
  - **–°—Ç–æ–ª–±—á–∞—Ç—ã–µ (`plt.bar`):** –ö–∞—Ç–µ–≥–æ—Ä–∏–∏, —á–∞—Å—Ç–æ—Ç—ã.
  - **–ö–æ–Ω—Ç—É—Ä–Ω—ã–µ (`plt.contour`):** –ü–ª–æ—Ç–Ω–æ—Å—Ç—å –∏–ª–∏ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –≤ 2D (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ñ—É–Ω–∫—Ü–∏–∏ –ø–æ—Ç–µ—Ä—å).
  - **3D-–≥—Ä–∞—Ñ–∏–∫–∏ (`Axes3D`):** –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è —Ç—Ä–µ—Ö–º–µ—Ä–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö.

#### ‚úçÔ∏è –ü—Ä–∏–º–µ—Ä —Å Matplotlib:
```python
import matplotlib.pyplot as plt
import numpy as np

# –õ–∏–Ω–µ–π–Ω—ã–π –≥—Ä–∞—Ñ–∏–∫ –∏ –≥–∏—Å—Ç–æ–≥—Ä–∞–º–º–∞
x = np.linspace(0, 10, 100)
y = np.sin(x)
plt.plot(x, y, label='sin(x)', color='blue')
plt.hist(y, bins=20, alpha=0.5, label='–†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ')
plt.title('–°–∏–Ω—É—Å–æ–∏–¥–∞ –∏ –µ—ë —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ')
plt.xlabel('X')
plt.ylabel('Y')
plt.legend()
plt.show()
```

**–ù–∞—Å—Ç—Ä–æ–π–∫–∞:** `plt.title()`, `plt.xlabel()`, `color`, `linestyle`, `plt.grid()`.

**–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:** –î–ª—è –¥–µ—Ç–∞–ª—å–Ω–æ–π –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ –∏–ª–∏ –Ω–µ—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –≥—Ä–∞—Ñ–∏–∫–æ–≤.

**–ó–∞–¥–∞–Ω–∏–µ:** –ü–æ—Å—Ç—Ä–æ–π –ª–∏–Ω–µ–π–Ω—ã–π –≥—Ä–∞—Ñ–∏–∫ –¥–ª—è `cos(x)` –∏ —Å—Ä–∞–≤–Ω–∏ —Å `sin(x)`.

## 4. ‚ú® Seaborn - –°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∞—è –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è

**Seaborn** ‚Äî –≤—ã—Å–æ–∫–æ—É—Ä–æ–≤–Ω–µ–≤–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –Ω–∞ –æ—Å–Ω–æ–≤–µ Matplotlib, —É–ø—Ä–æ—â–∞–µ—Ç —Å–æ–∑–¥–∞–Ω–∏–µ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏—Ö –≥—Ä–∞—Ñ–∏–∫–æ–≤.  
* –ò–º–ø–æ—Ä—Ç: `import seaborn as sns`.  
* –†–∞–±–æ—Ç–∞–µ—Ç —Å Pandas DataFrame.  
* –ì—Ä–∞—Ñ–∏–∫–∏ —ç—Å—Ç–µ—Ç–∏—á–Ω—ã "–∏–∑ –∫–æ—Ä–æ–±–∫–∏".

#### –û—Å–Ω–æ–≤–Ω—ã–µ –≥—Ä–∞—Ñ–∏–∫–∏:

- **–ü–∞—Ä–Ω—ã–µ –≥—Ä–∞—Ñ–∏–∫–∏ (`sns.pairplot`):** –ü–æ–ø–∞—Ä–Ω—ã–µ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏.
